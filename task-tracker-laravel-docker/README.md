# 🚀 Task Tracker (Dark UI + Docker)

## 📌 Description
A modern task tracker web app with dark theme, animations, and local storage support.

## ✨ Features
- Add tasks  
- Delete tasks  
- Data stored using localStorage  
- Animated dark UI  
- Docker support  

## ▶️ Run
php -S localhost:8000

## 🌐 Open
http://localhost:8000
