## 📌 Project Overview
A modern **Task Tracker Web Application** with a beautiful dark UI, smooth animations, and persistent task storage using localStorage.

This project is designed to demonstrate **frontend UI design + basic backend + Docker usage**.

---

## ✨ Features
- ➕ Add new tasks  
- 🗑 Delete tasks  
- 💾 Persistent storage using localStorage  
- 🌌 Animated dark theme UI  
- 💫 Smooth transitions & effects  
- 🐳 Docker support  

---

## 🛠 Tech Stack
- HTML  
- CSS (Animations + Glassmorphism)  
- JavaScript  
- PHP (Basic server)  
- Docker
